# Web-Ban-Do-Cong-Nghe

## This project is created using

- Client: ReactJS
- Server: Nodejs, ExpressJs
- Database: Mysql

---


## Features

- User

  - Login
  - Register
  - Verify Email
  - Forgot Password
  - My Account
    - Update info
    - View order history
    - Log out
  - Cart
  - Order products
  - Payment process
  - Write review

- Role-based authorization
  - User
  - Admin
- Products Management
  - CRUD
  - Filter, Order
- Chatbot
